Craftbot for Arduino


The Craftbot for Arduino is a binary service that runs on a Raspberry Pi, or other linux-based machine. To use it, your computer, tablet or phone need to be on the same network, connected with a cable or wireless network.

The in-line web server of Craftbot serves a web page, containing the web-based code editor Code Mirror alongside buttons for invoking remote functions; compiling and uploading code from the Arduinobot computer to a connected Arduino microcontroller, via a local MQTT broker over web sockets (ports 80, 443) and further configuration is picked up as a retained message on the topic config.

The service also holds a web server, written in Nim and scripts that communicate with a Arduino IDE running headlessly on the same computer, or on a remote device. More information on Craftbot for Arduino on Github github.com/ecraft2learn/arduinobot.
