Mathematica is a commericial programming language that is free and included in the standard Raspberry Pi release. It has facilties for communicating with the Raspberry Pi pins and with a connected Arduino. It has support for machine learning including a chapter in a high school text book with many project ideas. It also provides easy access to a large variety of open data sources. 3D models suitable for printing can also be created in Mathematica, though in many cases doing so requires a high-level of mathematical sophistication.

This could be a good programming tool for eCraft2Learn projects for those familiar with the Mathematica programming language.

The eCraft2Learn project provides no support or integration for this tool.
