Arduino IDE

The Arduino IDE is a free text-based Arduino programming system. There is both a web version and an app. C programs are compiled to run directly on an Arduino. This means the Ardunio can be embedded in an artefact and run without either being accompanied by a Raspberry Pi or in communication with another computer.

The Arduino IDE relies upon C, a textual language that was designed for use by professionals not students. ArduBlock is a blocks-based language designed for beginneers that has approximately the same functionality as the Arduino IDE.

*The eCraft2Learn project provides no support or integration for this tool.
