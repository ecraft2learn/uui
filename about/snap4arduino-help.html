Snap4Arduino

Snap4Arduino adds several new blocks to Snap! for the purpose of controlling Arduinos. It is documented here. Note that the web version runs only in Chrome and requires the installation of a Chrome extension. To do so unzip this file, go to chrome://extensions and drag and drop the folder there. This has already been performed if you are using an eCraft2Learn SD card on a Raspberry Pi.

The eCraft2Learn Snap! blocks for communicating with AI cloud services and machine learning require HTTPS support. Until the site supports HTTPS we are using a fully-functional copy of Snap4Arduino except that it cannot communicate with the Snap! cloud. Projects should be saved to the browser's memory or exported to the local file system.
