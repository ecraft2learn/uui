Tinkercad 3D Design

Tinkercad is a free web-based app for creating 3D models suitable for 3D printing. You donÂ´t need previous experience with CAD (computer-aided design) to create and print impressive 3D models. Users create models via the Tinkercad interface, in constrast to BeetleBlocks which supports creating programs that construct 3D models. The main functions you can use are: 1) Place basic building blocks that can be added or removed (users can also import their own forms or work with existing ones). 2) Adjust, move and/or rotate shapes freely in space. 3) Combine or group together a set of shapes to create detailed templates.

Tinkercad can also be used to construct electronic circuits connected to Arduinos and simulate their running (check the next Tinkercad tool).

Note that TinkerCad requires that one log in before using this tool.
