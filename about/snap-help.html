Snap!

Snap! is a programming language â€” a notation in which you can tell a computer what you want it to do. Unlike most programming languages, though, Snap! is a visual language; instead of writing a program using the keyboard, the Snap! programmer uses the same drag-and-drop interface familiar to computer users.

Snap! is a big brother of Scratch. It can do much more than Scratch but is a bit more complex. A Reference manual is documented here.

Most importantly it has enabled the eCraft2Learn project to add new blocks. These include

  1. the ability to use speech input and output
  2. image recognition by connecting to AI cloud services (free API keys required)
  3. the ability train the system to distinguish images from the camera
  4. train machine learning models and create new ones.
