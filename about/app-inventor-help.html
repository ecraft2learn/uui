MIT App Inventor

MIT App Inventor is a free blocks-based programming language for Android tablets and smart phones. Authoring Apps takes place in a browser on a PC. It is documented here. Apps can be constructed that communicate via BlueTooth or Wifi to a Raspberry Pi or an Arduino. Raspberry Pis include BlueTooth and Wifi while some Arduinos do and a shield can be added to add such functionaliy.

The eCraft2Learn project provides no support or integration for this tool.
