20 When you turn on the Ultimaker 2+ for the first time, the display shows “Welcome”. The Ultimaker 2+ will guide you through some steps for calibrating the build plate. Follow the instructions shown in the display. The Ultimaker 2+ will do the “homing” first. This means that it will move the print head to the left back corner and the build plate towards the bottom, in order to set the origin point. After this, you can start the bed leveling. If you do not see the configuration wizard, navigate to “Maintenance” > “Advanced” and confirm “Factory reset”. FIRST USE
